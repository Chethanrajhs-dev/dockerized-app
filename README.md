# Dockerized Application

## 📌 Project Overview
This project demonstrates how to containerize an application using Docker.

## 🚀 Tools Used
- Docker
- Linux

## ⚙️ Workflow
1. Create application file
2. Write Dockerfile
3. Build Docker image
4. Run container

## 📂 Features
- Lightweight containers
- Easy deployment
- Platform independent

## 🛠️ Commands
- docker build
- docker run

## 📸 Output
Application successfully running inside Docker container.
